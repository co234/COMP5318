limitations of Perceptron

Perceptron cannot solve XOR problem (linear classifier cannot handle non-linear patterns)

 

MLP (Multi-Layer Perceptron)

  1. Backpropagation
  2. Hidden layers (adding more neurons in between the input and output layers)

Universality theorem (Hecht-Nielsen 1989): "Neural networks with a single hidden layer can be used to approximate any continuous function to any desired precision."

 

Activation functions

  1. sigmoid

    • vanishing and exploding gradients
    • output are not 0-centred

  2. tanh

    • 0-centred
    • vanishing and exploding gradients

  3. ReLU

    • avoid vanishing and exploding gradients
    • some units can be fragile during training and die

  4. Leaky ReLU

    • fix ReLU problem

 

Regularisation

  1. Dropout
  2. Early stopping
  3. Data augmentation

 

Initialisation

  1. All 0

    • Output, gradients, parameters update the same, no difference between all neurons.

  2. Random initialisation

    • Vanishing and exploding problem

  3. Calibrating the variances

    • Xavier Initialisation

      • variance for input and output are same for all layers
      • still have vanishing and exploding problem when deeper (use sigmoid)

    • He initialisation

      • Relu

 

Optimisation

  1. Gradient descent

    • difficult to choose learning rate
    • we don't want to update all of data and features to the same extent
    • Easily get trapped in saddle point.

  2. Momentum:

    • Momentum tries to accelerate SGD in the relevant direction and dampens oscillations. So gradients at same directions, increase update; gradients change directions, reduce update.

  3. NAG:

    • Make a big jump in the direction of the previous accumulated gradient, then measure the gradient where you end up and make a correction

  4. Adaptive learning rate methods:

    Adapt learning rate based on feature importance and frequency.

    • Adagrad
    • Adadelta
    • RMSProp
    • Adam

Overall: Momentum and NAG run faster in the relevant direction. RMSProp, Adadelta, Adagrad, Adam more easily escape saddle point.
